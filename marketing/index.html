// KEY FEATURES

Unified app management

Every Axi app in one place. Each card shows installed version, latest release, and a single action button that does the right thing.

Automatic update detection

On startup, AxiOM checks GitHub for each app and compares against what's installed. A gold badge appears on the tray icon when updates wait.

Native install & update

On Windows, the NSIS installer runs — UAC, install path, shortcuts handled. On Linux, AppImages land in ~/Applications/ ready to go.

Gear Lever integration

On Linux, AxiOM detects Gear Lever and hands off AppImages for full desktop integration. Not installed? It offers Flatpak install inline.

Tray-first, always there

Close the window — AxiOM keeps running. One tray click and your launcher is back. Set it to start on login and forget it's there.

Consistent Axi design

Dark canvas. Gold accents. Cinzel titles. AxiOM matches the look and feel of every app it manages — and stays out of the way the rest of the time.

// FAQ
Do I need to install all the apps?

No. AxiOM shows every Axi app whether it's installed or not, so you can install just the ones you need. The ones you skip are just one click away if you change your mind.

Does AxiOM run in the background?

Yes — it lives in the system tray. Close the popup and AxiOM keeps running so the tray icon stays available. You can enable start-on-login from settings.

Can I use AxiOM on Linux?

Yes. Linux installs land as AppImages in ~/Applications/. If Gear Lever is installed, AxiOM hands them off for full desktop integration. If not, AxiOM offers to install Gear Lever via Flatpak.

Where do downloads come from?

The official GitHub Releases page for each app. AxiOM never proxies or rehosts binaries — it downloads directly from github.com/darkharasho/<app>/releases over HTTPS.

How often does it check for updates?

Every 30 minutes in the background, plus a quick re-check the first time you open the tray after 5 minutes of inactivity. You can also hit Check for updates any time from the popup.

Is it free?

Yes. GPL-3.0 licensed. Free forever, no paid tier.
